# CS 开箱模拟器

一个纯前端的 CS 风格开箱模拟小软件，包含滚轮开箱动画、StatTrak™ 工艺、磨损与估值、出货率统计、成就系统等功能。

## 功能特性
- 滚轮开箱动画与指针停靠
- StatTrak™ 工艺显示与溢价
- 五段磨损影响价格
- 开箱记录与统计（含出货率）
- 成就系统（首次开箱、稀有度、连胜等）
- 多箱子选择

## 快速开始
直接打开 `index.html` 即可运行，无需安装任何依赖。

## 目录结构
- `index.html` 页面结构
- `style.css` 样式与动画
- `script.js` 逻辑与数据

## 备注
这是一个娱乐性质的模拟器，不涉及真实物品或交易。
